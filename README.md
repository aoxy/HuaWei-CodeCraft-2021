## 运行方式

```bash
# 克隆仓库进入项目
git clone git@github.com:aoxy/HuaWei-CodeCraft-2021.git
cd HuaWei-CodeCraft-2021/SDK_C++/

# 编译源码
sh ./build.sh

# 运行程序
./bin/CodeCraft-2021

# 编译并运行
sh ./build_and_run.sh

# 打包提交
./CodeCraft_zip.sh

# 测试运行（结果输出到sampleout.txt文件）
./bin/CodeCraft-2021 < ../samplein.txt > ../sampleout.txt
# 或者
sh ./test.sh

# 大规模数据测试（结果输出到training-*out.txt文件）
sh ./train.sh
```
